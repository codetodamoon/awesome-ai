# AI Collections

个人AI资料收集库，用于存放和管理各类AI工具的Prompts、Skills、Agents和Gems。

## 项目结构

```
ai-collections/
├── chatgpt/          # ChatGPT相关资源
├── gemini/           # Gemini相关资源
├── claude-code/      # Claude Code相关资源
└── common/           # 通用资源
```

## 资源分类

每个平台目录下包含以下子分类：

- **prompts/**: 提示词模板
- **skills/**: 技能定义
- **agents/**: 智能代理配置
- **gems/**: 其他有价值的资源

## 使用说明

1. 选择对应的平台目录（chatgpt/gemini/claude-code/common）
2. 选择资源类型（prompts/skills/agents/gems）
3. 查看或添加相应的资源文件
